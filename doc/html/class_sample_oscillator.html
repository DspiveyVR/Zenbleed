◆ loadSampleFromFile()

void SampleOscillator::loadSampleFromFile ( const juce::File &  file)

Loads an audio sample based on a given file path.

This method is called when the user selects a file and also when the plugin is first loaded by the DAW (assuming a sample has been loaded previously).

Warning
This method should only be called from the message thread since File I/O is not a realtime operation.
